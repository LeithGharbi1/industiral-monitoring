# Smart Manufacturing Data Platform

## 🎯 Objective
This project simulates an end-to-end industrial data engineering pipeline for manufacturing production monitoring.

It includes:
- Data generation (machine simulation)
- Data ingestion layer
- ETL processing
- Big Data processing (Spark)
- Orchestration (Airflow)
- Dashboarding (Power BI / Streamlit)
- Cloud-ready architecture

---

## 🏗️ Architecture Overview

Data Source → Ingestion → ETL → Storage → Spark Processing → Dashboard

## ⚙️ Tech Stack

- PostgreSQL
